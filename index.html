resume

Hello, my name is Vivian.

I'm a software developer based in NYC.

I graduated class of 2020 from Fordham University, with a B.S. in Computer Science and Mathematics and a minor in Bioinformatics. Growing up I moved all around the country, seeing many different places and meeting people from allover and I've always had a curiosity for the sciences and technology. All this has informed my passion for helping others and my eventual goal of working on an M.S. in computational biology. As tech continues to change and evolve it is more important than ever, I feel, to be ethical and do no harm.

Strainer

Strainer is a chrome extension that was designed to act as a webfilter. It uses jquery to match words on a site to a blacklist and then renders the parent container invisible. I never finished it, but as my first hackathon project it holds a special place for me.

Smart Flights

A project for StudentHackV in Manchester, England. Smart Flights is a project that attempted to make finding cheap travel easier. By finding the cheapest series of flights between continents using the skyscanner API, it would find the cheapest reguardless of connections.

Alexa Skills

These Alexa skills were made as novelty because I think Alexa's dead pan is funny and Amazon was giving away an Echo if you published two. I quickly learned to implement Amazon's services in the creation of skill that gained a modest user base.

Get Bazaar

This project was made for my most recent hackathon, Hack Cooper in New York City. Get Bazaar was a proximity marketing progressive web app. As the user moved into the proximity of a local buisness, they would be able to cupons and announcements from enrolled clients. It placed fourth overall in the competition.
